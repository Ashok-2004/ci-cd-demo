# EmployeeHub Frontend

This folder contains the React, TypeScript, Vite, Tailwind CSS, Axios, and Vitest frontend for EmployeeHub Demo.

## Why This Folder Exists

The frontend is the user interface. In later phases, it will show login, register, dashboard, and employee management pages.

## How It Connects

React components render the UI. Frontend services will use Axios through `src/api/apiClient.ts` to call the ASP.NET Core API in the `backend` folder.

## Useful Commands

```powershell
npm run dev
npm test
npm run build
```
